# Community Evaluations Index

This index tracks all SPERB evaluations submitted by the community, beyond the inaugural pilot benchmark.

---

## How to Submit

1. Complete the [Evaluation Template](../scoring/EVALUATION_TEMPLATE.md)
2. Save your file as `evaluations/[handle]-[YYYY-MM].md` in this directory
3. Add one row to the table below
4. Open a Pull Request with title: `[EVAL] Entity Name (@handle) — SPERB v1.0 — YYYY-MM`

Submissions are reviewed for rubric adherence and evidence quality before merge.

## Evaluation Standards

Every submitted evaluation must:
- Use the official [Evaluation Template](../scoring/EVALUATION_TEMPLATE.md)
- Score all 8 dimensions with evidence citations
- Declare any conflict of interest
- Use SPERB v1.0 (or specify version if later)

Evaluations without evidence justifications will not be merged.
